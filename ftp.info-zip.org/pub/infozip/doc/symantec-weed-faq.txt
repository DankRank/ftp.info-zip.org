> Sehr geehrter Herr Lenz, Im folgenden Auszug finden Sie mehr Informationen 
> wegen dieses Weed Virus:
> 
> The Weed Virus Is Detected when Booting the Computer with the Emergency Disk
> 
> Module(s):  CLF
> Operating System(s):  Windows 95, Windows 98
> 
> Situation:
> 
> You boot the computer from the Emergency disk provided with Norton Anti Virus 
> version 5, and the NAVDX DOS scanner detects a file infected with the Weed 
> virus. However, the NAVW32 Windows scanner does not detect this infection.
> 
> Solution:
> 
> The definitions on the Emergency disk do not contain the latest virus 
> definitions, and this results in the false detection of the Weed virus. 
> You should create a Rescue disk set with current definitions.
> 
> This was fixed in the 5.02 release, available on the 3.04 Solutions CD.
>  
> Internal Notes:
> 
> The NAV Emergency Boot Disk that has been shipping since August 1998 had this 
> problem. This is the one that shipped with NAV 5.0 (Atomic) and later NAV 5.x 
> versions, NAV Solution CD 3.02 & 3.03, and whatever other products the NAV 
> Emergency Boot Disk ships with.
> 
> The good news is that we just fixed it. The new disk that does not have this 
> problem is now in manufacturing and customers should get it with the above 
> products & future products. There is no problem with the product itself -
> just with the Emergency Boot Disk.
